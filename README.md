# Anti Thavakai (NextGen Chat)

A full-stack real-time chat application inspired by Discord and WhatsApp.

## features
- Real-time messaging
- Group chats
- Voice & Video calling (WebRTC)
- Neonic/Cyberpunk UI

## Structure
- `/client`: Next.js 14 Frontend
- `/server`: Express + Socket.io Backend

## Setup

### Server
```bash
cd server
npm install
npm run dev
```

### Client
```bash
cd client
npm install
npm run dev
```

## Environment Variables
Create `.env` in `server` and `client` directories based on example.
